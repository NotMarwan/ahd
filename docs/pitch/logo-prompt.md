# Ahd logo — open creative prompt (v2, copy verbatim)

v2 per Marwan: the model is FREE — no imposed palette, no imposed shape. It must
propose THREE distinct, varied directions. Only the essence is fixed.

```
You are a world-class brand identity designer. Design a logo for "عهد" (Ahd).

WHAT AHD IS (the only constraints — everything else is yours):
- A Saudi fintech where the bank WITNESSES good-faith loans between people —
  it never lends, never judges, never charges interest. Think: a trusted
  neutral witness, a kept word, a sealed document.
- The wordmark is the Arabic word «عهد» (meaning: covenant / promise).
  Arabic script only — no Latin letters in the mark.
- Feeling to evoke: trust, dignity, permanence, warmth. Saudi-rooted but
  modern — NOT a cliché (no crescents, no coins, no generic bank pillars).
- Must be AAA quality: minimal, timeless, legible at favicon size (24px),
  works in one color, works on light and dark.

YOUR TASK — propose THREE genuinely DIFFERENT directions, not variations of
one idea. For each direction, choose everything yourself: the concept, the
shape language, the color palette (any colors you believe in), the type
treatment of «عهد», and the cultural reference (if any). Surprise me — I want
to see range: e.g. one could be calligraphic, one geometric/abstract, one
emblem/seal-like — but don't limit yourself to these examples.

For EACH of the three directions deliver:
1. The mark alone, centered, transparent background, 1:1 ratio.
2. A one-line rationale (why this shape, why these colors).
3. A small preview of it at tiny size next to full size.

Rules: flat vector aesthetic, no 3D, no drop shadows, no stock-icon look.
Generous negative space. Each direction must stand on its own as a real
bank-grade identity.
```

After he returns 3 directions: pick one (or ask for a round-2 refinement of the
winner), then save as `promo/public/logo/ahd-mark.png` (transparent 1:1) and set
`USE_LOGO_ASSET = true` in `promo/src/sadu.ts` → `npm run render:sadu`.
