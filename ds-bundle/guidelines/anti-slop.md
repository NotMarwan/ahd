# Ahd anti-slop charter — ميثاق ضدّ السلوب

Distilled from hallmark (3.6k★) + taste-skill (61k★) + soft-skill into Ahd's context. Every design
produced in this project — by any agent — follows this or gets revised.

**Design read (locked):** dignified Saudi Islamic-fintech for a judging panel; trust-first premium,
Editorial-Luxury archetype. Dials: `VARIANCE 7 · MOTION 3 · DENSITY 3`.

## BANNED — instant fail

- AI-purple/mesh gradients · centered hero over dark mesh · three equal feature cards · generic
  glassmorphism · infinite looping micro-animations · the Inter+slate-900 default look.
- **Invented numbers.** Metrics come ONLY from the live gate banner or `EVIDENCE-BRIEF.md` (graded,
  vintage-labeled). No "+47%", no "trusted by 50k", no fake testimonials — ever.
- Fake browser bars / phone frames / code-window chrome (real screenshots in a `<figure>` instead).
- Italic headings, or an italic emphasis word inside a heading. Emphasis = weight or gold accent.
- Inline hex/rgb bypassing tokens — every color and font-family references a `var(--*)` from
  `styles.css`. New value needed → add a named token first.
- Spine violations (see README): % glyphs in product stats · red for lateness (red = tamper only) ·
  any numeric trust signal · missing «(محاكاة)» on simulated rails.

## REQUIRED texture — the Ahd flavor of Editorial Luxury

- **Two moods only:** the warm cream field (`--bg`) for living surfaces; the ceremonial seal-green
  dark field (`--seal-bg` + gold hairline frame + geometry watermark) for proof/title moments.
- **Macro-whitespace.** Sections breathe (double the instinctive padding). Eyebrow pill tag
  (uppercase-tracking equivalent: small gold kicker line) before major headings.
- **Double-bezel cards.** Premium card = outer hairline shell + inner core with concentric radii —
  never a flat card on a flat background.
- **Soft diffused shadows** via `--shadow-sm/--shadow/--shadow-lg` — never harsh `rgba(0,0,0,.3)`.
- **Motion: few and physical.** Entry = gentle fade-up 500–800ms `cubic-bezier(0.32,0.72,0,1)`;
  always guarded by `prefers-reduced-motion`. No motion for decoration's sake.
- **Structural variety.** No two surfaces share the same hero→cards→CTA rhythm. Vary the macro
  structure per surface (bento, split, cascade) while keeping tokens identical.
- **Typography:** Arabic-first RTL, system stack (product runs offline — no CDN fonts). Hashes and
  commands in `--mono`, LTR. Scale contrast: one huge display moment per surface, everything else calm.
