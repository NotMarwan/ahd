# StatTiles — بطاقات إحصاء

Two-column stat grid for aggregate numbers.

- `.tiles` (2-col grid) → `.tile` → `.tl` label, `.tv` value (with `<small>` unit), `.tc` caption.
- Spine rule: aggregates only — never an individual's number, never a percentage glyph (٪/%) in product copy; write «من 9» style fractions instead.
- Money is integer SAR (halalas internally); never float.
