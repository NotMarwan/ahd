# Buttons — الأزرار

Button classes are SCOPED to their parent wrappers — always pair them:

- `.cr-act .primary` / `.ol-act .primary` — full-weight teal action (seal, pay).
- `.ol-act .ghost` — gold-soft mercy action («اجعلها صدقة» — the warmest button in the product).
- `.cr-trybar .ghost` — neutral trial actions.
- `.navbtn` (+`.on`) — pill nav; `.fchip` (+`.on`) — filter chips; `.dask` — teal-soft call-to-ask pill.
- Never invent a bare `.btn` class — it does not exist in this system.
- Simulated integrations carry «(محاكاة)» in the label — keep it (honesty rule, OT-VAL).
