# TabsChips — تبويب وترشيح

- `.tabs` segmented control on `--mute-soft`; active `.tab.on` lifts white with `--shadow`.
- `.dfilter` row of `.fchip` pills; active chip inverts to ink.
- `.hsummary` — centered muted summary card; bold values inherit ink.
- Keep tab labels to one word; RTL order is source order.
