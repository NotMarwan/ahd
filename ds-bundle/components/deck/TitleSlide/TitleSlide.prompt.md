# TitleSlide — شريحة العنوان

The ceremonial 16:9 title slide: sealed-document field, gold-gradient wordmark, one hairline ceremonial frame, and a REAL proof line (live gate banner + tripwire sha — recompute before any showing, never copy).

- Field: `--slide-bg` seal-green with two soft radial glows; geometry watermark = the product's own `.hero::before` idiom at slide scale.
- Wordmark: gold gradient text — never flat yellow, never an image, never italic.
- Discipline (hallmark): all colors via locked tokens · honest copy only (no invented metrics) · no fake chrome · headings roman.
- Deck slides derive from this: same field, same frame, content swaps. Keep the proof line only on title + closing slides.
