# 🎙️ العرض والمسرح — ٣ دقائق

> [!tip] التجهيز قبل الصعود (٣ نوافذ)
> ١) التطبيق على **«حافظة الإثبات»** لعهدٍ مختوم — المسار: دفتري ← صفّ سلطان ← **⋯** ← «وثيقة الإثبات 🔏»
> ٢) طرفيّة فيها لافتة **`AHD GATE ✅ 1687/0`** الخضراء (شغّلتها قبل دقائق)
> ٣) العرض المجمَّد خلفهما للاحتياط

## البيتات السبعة (النصّ الحاكم: `docs/pitch/script-ar.md`)

| ⏱ | البيت | الأزرار بالضبط ✓ |
|---|---|---|
| 0:00–0:25 | **الافتتاح البارد: العبث الحيّ** ← «✗ عبثٌ مكشوف» ← «بنكٌ يَشهد ولا يُقرض» | «جرّب العبث بالمبلغ 🧪» ← «أصلِح الوثيقة» |
| 0:25–0:45 | سطر المشكلة المضغوط (أثناء التنقّل) | — |
| 0:45–1:15 | الورع: الفاحص يمنع — «حارسان» | «🧪 شرط غرامة» ← «أزِل الشرط وأعد الصياغة» ← «اختم العهد عبر نفاذ» |
| 1:15–1:45 | الرحمة: بلا موعد + إبراء | «🤍 اجعلها صدقة» ← «أُبرئ المبلغ كاملًا — لوجه الله» |
| 1:45–2:15 | ⭐ المقاصّة ٩←٢ + «المال المتحرّك ١٬٨٠٠⟶٩٠٠» | تفتح محسوبة — أشِر |
| 2:15–2:40 | الأرقام (٥٨٫٦٪ · ١١٥ مليار «٢٠٢٠–٢١» · **١٬٦٨٧**/٠) + لمحة الطرفيّة | لمحة ثانيتين |
| 2:40–3:00 | الإنماء + «مَن يُعرّف الفئة يملكها» + الطلبات + الجملة الروحيّة | العودة للرئيسيّة |

> كلّ نقرةٍ فوق **متحقَّقة فعليًّا 16/16** في متصفّح حقيقيّ (8 يوليو).

## مقاعد الأسئلة الجاهزة
- «وايش يحمي الأطراف؟» ← افتح **«الضمانات والحدود» 🧭**: كلّ بندٍ وتحته «يحرسه: <ملفّ اختبار>»
- «أين تحليل البيانات؟» ← افتح **«أثر عهد» 📊** + بطاقة Q-H1
- «بنيتوه قبل الهاكاثون؟» ← نعم وبفخر — Q-H2
- أصعب ٦ أسئلة ببطاقات: `docs/pitch/top6-cards-ar.md`
