# عهد · الديوان — واجهة الويب لسطح المكتب

صفحة واحدة (`index.html`) تستعرض **كل خدمات عهد** ببيانات تجريبية حتمية على حساب مستعار (نايف العتيبي)، بهوية «نسيج الثقة»: الأزرق خيط قائم، الذهب وفاء، الكهرمان تنبيه، والأحمر محجوز لكسر الختم وحده.

**رابط التشغيل المباشر:** https://notmarwan.github.io/ahd/webapp/

## التشغيل

```bash
# مباشرة (بلا خادم):
start webapp/index.html          # ويندوز — يعمل عبر file:// كاملًا

# أو عبر خادم محلي:
cd webapp && npx serve .         # أي خادم ثابت يكفي — لا توجد اعتماديات
```

- **دون إنترنت بالكامل**: الخطوط محلية (`fonts/`)، لا CDN، لا fetch.
- **حتمية**: تاريخ ثابت `AS_OF 2026-06-21`، لا `Date.now`، لا عشوائية. يحرسها `tests/app/webapp-showcase.test.cjs` (45 تأكيدًا ضمن البوابة).
- **وضع الالتقاط**: `index.html?static=1` يعطّل الحركة ويثبّت الحالة النهائية (للقطات التوثيق ولقطات headless).

## ماذا في الصفحة؟

12 قسمًا: نظرة عامة (وش الوضع + النسيج الحي) · دفتري (+ تصديق السداد + قيد اليوم) · المقاصة (9→2 + رِفْق + رضا الأطراف) · الجمعية والدائرة (+ المسودات والتخريج) · سُلف وقروض مفتوحة · وثيقة الليل (الختم + العبث + التحقق المستقل) · الرعاية والخلاف · إشارة الوفاء · الأثر والمصادر · الأجرة والخطط · الحدود والأساس الشرعي (+ فاحص الربا الحي) · فهرس القدرات (كتالوج الجوال الـ19 حرفيًا) · الإعدادات.

تفاعلات حية: تبويبا دفتري، تصفية الأشخاص، بحث الشريط، مفتاح رِفْق (يعيد حساب المقاصة)، رضا الأطراف قبل ختم التسوية، تجربة العبث (الأحمر الوحيد)، فاحص الربا، أرقام غربية/هندية، إخفاء المبالغ.

كل زر محاكاة موسوم «(محاكاة)» في نصه. أرقام السوق موسومة مُقاس/توضيحي كما في `app/features/sources.js`.

**لا يلمس** `demo/` ولا `app/` — سطح عرض مستقل يستهلك نفس بذور البيانات والدلالات.
